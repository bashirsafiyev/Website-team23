Hour Of Code

Code.org originally introduced Hour of Code from December 9 to December 15, 2013, during Computer Science Education Week. The goal of the Hour of Code is to get people interested in coding and familiar with it. In order to accomplish this, hour-long, ostensibly interesting, and instructional activities are hosted. Hour of Code aims to create computer science classes, motivate students, and educate teachers to teach computer science, among other things. A number of educational online learning portals also supported it. Over 600 million lines of code were written by roughly 20 million participants. Typically, Hour of Code begins on the second Tuesday in December and ends on the third Tuesday of the same month.
